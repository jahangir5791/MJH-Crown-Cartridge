# Unity Project Settings

These files are auto-generated by Unity. Do not edit manually.

## Important Files

- `ProjectSettings.asset` - Main project settings
- `PlayerSettings.asset` - Platform-specific settings (Android/iOS/PC)
- `GraphicsSettings.asset` - Graphics pipeline
- `InputManager.asset` - Input configuration
- `QualitySettings.asset` - Quality levels
- `BuildSettings.asset` - Build configuration
